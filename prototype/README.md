# Computer Use Verifier Kit Prototype

Source: [[UseDesktop as Verifier Infrastructure for Computer-Use Agents]]  
Package: [[Computer Use Verifier Kit]]

## What this demonstrates
A small, fillable artifact that turns the X signal into operator evidence instead of a passive note.

## How to use
1. Start with [[Computer Use Verifier Kit/Trial Operator Handoff Checklist]] so the trial runner follows the existing packet → evidence → decision → debrief sequence.
2. Open [[Computer Use Verifier Kit/Computer Use Verifier Packet]].
3. Fill the scenario, inputs, constraints, evidence, and decision fields.
4. Grade the captured proof with [[Computer Use Verifier Kit/Verifier Evidence Quality Rubric]] before changing validation claims.
5. Attach logs/screenshots/repo links before claiming the idea worked.
6. Fill [[Computer Use Verifier Kit/Verifier Promotion Decision Card]] before promoting the skill draft or calling the verifier reusable.

## Next iteration ideas

- Use the promotion card to choose promote / pilot-only / iterate / hold from evidence, not memory.
